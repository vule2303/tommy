Câu 1: Trình bày sự khác nhau giữa Inline, Inline-block và block của display. Ví dụ minh họa

Inline : các item có kiểu display: block sẽ không thể set width và height chỉ có thể điều chỉnh margin và padding left and right

Block : Các item có kiểu display: block sẽ set được width, height, margin, padding đầy đủ 4 hướng (top, bottom, right, left) và luôn được xuống dòng và chiếm toàn bộ width nếu width không được set

Inline-Block : kiểu display: inline-block sẽ được sắp xếp giống với kiểu display: inline, nghĩa là các items sẽ được xếp cùng nhau trên một dòng . Tuy nhiên các items sẽ có thuộc tính của display: block như là có set width, height, margin, padding đủ 4 hướng.

Ví dụ minh họa:

            
<!DOCTYPE html>
<html>
<head>
<style>
span.a {
  display: inline; 
  width: 100px;
  height: 100px;
  padding: 5px;
  border: 1px solid blue;  
  background-color: yellow; 
}

span.b {
  display: inline-block;
  width: 100px;
  height: 100px;
  padding: 5px;
  border: 1px solid blue;    
  background-color: yellow; 
}

span.c {
  display: block;
  width: 100px;
  height: 100px;
  padding: 5px;
  border: 1px solid blue;    
  background-color: yellow; 
}
</style>
</head>
<body>

<h1>VÍ DỤ MINH HỌA</h1>

<h2>display: inline</h2>
<div> 
    đây là ví dụ<span class="a">display:</span> <span class="a">inline</span> 
</div>

<h2>display: inline-block</h2>
<div>
    đây là ví dụ<span class="b">display:</span> <span class="b">inline-block</span> 
</div>

<h2>display: block</h2>
<div> 
    đây là ví dụ<span class="c">display:</span> <span class="c">block</span> 
</div>

</body>
</html>
Câu 2: Trình bày sự khác nhau giữa các thuộc tính position(relative, absolute, fixed, sticky).Ví dụ minh họa

Position sticky: Phần tử được định vị bằng Position sticky sẽ thay đổi vị trí dựa trên thao tác cuộn của người dùng. Với Position sticky, phần tử sẽ linh động chuyển từ trạng thái relative sang fixed và ngược lại

Ví dụ code:

                  
<!DOCTYPE html>
<html>
<head>
<style>
div.sticky {
  position: -webkit-sticky;
  position: sticky;
  top: 0;
  padding: 5px;
  background-color: #cae8ca;
  border: 2px solid #4CAF50;
}
</style>
</head>
<body>

<p>Thử <b>cuộn</b> xuống!</p>


<div class="sticky">Đây là sticky!</div>

<div style="padding-bottom:2000px">

  <p>....</p>
  <p>....</p>
</div>

</body>
</html>

Position relative: Phần tử được định vị bằng Position relative sẽ nằm ở vị trí tương đối so với vị trí static. Bạn có thể sử dụng các CSS như: Top, Bottom, Left, Right để định vị vị trí của phần tử đó.

Ví dụ code:

                    
<h3>{position: relative;}</h3>
<div style=”position: relative; width:250px;”>                       
position: relative; width:250px;                       
</div>                        
<br>                      
<h3>{position: relative; left:50px; top:50px;}</h3>                       
<div style=”position: relative; left:50px; top:50px; width:250px;”>                       
position: relative; width:250px; <br/>                       
left:50px; top:50px;                       
</div>

Position fixed: Phần tử được định vị bằng Position fixed sẽ có vị trí tương đối so với Viewport trên trình duyệt của bạn.

Ví dụ code:

                  
<!DOCTYPE html>
<html>                  
<head>                  
<meta charset=”utf-8″/>                  
<title>CSS Position</title>                 
<style>
div {                 
background-color:yellow;                 
padding: 5px;                 
}
</style>                  
</head>                  
<body>                 
<h2>{position: fixed}</h2>               
<div style=”position:fixed; bottom:20px; right:15px;”>               
position:fixed; bottom:20px; right:15px;                 
</div>                  
<p>Content .. 1</p>                  
<p>Content .. 2</p>                  
<p>Content .. 3</p>                  
<p>Content .. 4</p>                 
<p>Content .. 5</p>                
</body>                 
</html>

Position absolute: Đối với một phần tử được định vị bằng Position absolute, phần tử sẽ tự động ưu tiên cho phần tử cha, ông gần nhất của chúng có chứa Position relative. Khi tìm thấy phần tử chứa Position relative, phần tử được định vị bằng Position absolute sẽ tự động tạo vị trí tương đối so với phần tử đó. Nếu không, phần tử định vị bằng Position absolute sẽ lấy vị trí tương đối so với Viewport trên trình duyệt của bạn

Ví dụ code:

                      
<!DOCTYPE html>
<html>                     
<head>                     
<meta charset=”utf-8″/>                     
<title>CSS Position</title>                      
</head>                     
<body>                     
<h2 class=”mb-3″>CSS {position:absolute}</h2>                     
<div style=”position:relative; height:200px;border:1px solid red;”>                     
Tui là dòng div {position:relative} (Đỏ).                     
<div style=”border:1px solid green;”>                     
Tui là dòng div thông thường (Xanh lá).                     
<div style=”position: absolute; bottom:10px; right:15px; background:yellow;”>                      
position: absolute; bottom:10px; right:15px;                     
</div>                     
</div>                      
</div>                     
</body>                     
</html>
